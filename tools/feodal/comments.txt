Библиотека для работы с файлами карт (.feods) и шаблонов для генерации карт (.worldbox) игры "Междоусобица" (нерусс. "The Internecine strife").

Модули:
    constants - содержит классы-перечисления, такие как Environments, Ages.
    lands - загрузчик базовых типов ландшафта (файл landscape.json).
    MapBackgroundGenerator - рисует изображение карты, яркостью подчеркивая высоту местности.
    MapTemplate - содержит классы MapTemplate (шаблон карты) и TemplateToMap (генератор карт по шаблону).
    tools - содержит вспомогательные классы:
        Tools - средства этапа генерации;
        MapBackgroundGenerator - генератор изображений ландшафтной карты;
        Picturization - обертка над MapBackgroundGenerator для сохранения изображений;
        AreaAgregation - средство обработки пересечения областей слоев в шаблоне карт.
    pngs - набор низкоуровневых функций для генерации файлов PNG.
    pathfinding - поиск пути. Используется для добавления на карту слоя коммерции и торговли.
    feods - набор функции для загрузки/сохранения данных в форматах карт (.feods) и шаблонов (.worldbox).
    generators - набор классов:
        - семейство LayerGenerator - отдельные слои карты, позволяющие без шаблона строить карты различной степени достоверности и сложности;
        - MapGenerator - объединяет все слои и выдает так называемый "пакет" - словарь со всеми данными новой карты;
        - PatternGenerator - генерирует "пакет" с данными нового шаблона. Может строить шаблон по картам;
    typical - набор базовых классов генераторов, являющихся фасадами для MapGenerator. Являются рабочими примерами конструирования генератора карты.
"""

typical
u"""
Набор фасадов (готовых сборок) MapGenerator. Являются рабочими примерами конструирования генератора карты.

Каждый фасад использует свой набор генераторов слоев. Последние делятся на:
    - генераторы карт высот;
    - генераторы базового ландшафта;
    - генераторы поселений/популяций;
    - генераторы распределения столиц;
    - генераторы вооруженных сил;
    - генераторы базовых границ областей (объединение незанятых клеток);
    - генераторы предистории областей (изменение границ и поглощение областей друг другом);
    - генераторы антропоморфного ландшафта (постройки, поселения, хозяйства и шахты).
Планируется ввести следующие слои:
    - генератор слоя коммерции (торговые пути);
    - генератор богатств (накопленные ресурсы клеток и областей за время предистории);
    - генераторы слоя благополучия (сколько раз клетки подвергались нападениям);
    - генераторы карты событий (слой событий - это список ID событий, номер позиции на "текстуре" - номер хода);
    - генераторы эпидемиологической обстановки (какая где болезнь на начало игры).

Набор содержит следующие фасады генераторов:
    - SeparatedCells - простая карта со сглаженным рельефом и областями размером с одну клетку.
    - SeparatesWithoutSea - тоже самое, что и SeparatedCells, но без водных клеток.
    - RandomicColonies - случайным образом распределяет людей по сглаженной карте и собирает большие области без предыстории.
    - Voolcano - моделирование вулканического острова с группой диких племен с их владениями, без предыстории.
    - Canderra - моделировние озера в кратере потухшего (супер) вулкана с дикими племенами, так же не воевавшими ранее.
    - Classic - феодальная грызня, при которой часть областей была упразднена. Отличается хорошим выбором столиц и мест для поселения.
"""
